# Battlebit Forge

Create and share loadouts

## Release

**Requires changesets**  
You can check the current release with `yarn changeset:status`.

**Update package versions:**  
yarn changeset:version

**Commit the changes:**  
git add .  
git commit (make sure to prefix with: `release`)

**Generate git tags:**  
yarn changeset:tag 

**Push the release:**  
git push --all
